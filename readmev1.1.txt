Projet : DigiRelik – Hexekta
Version : V1.1
Date : 24/05/2025

Description :
Cette version intègre une première structure fonctionnelle pour l’exposition et la préparation des artefacts au mint NFT (eCash – XEC) et à l’horodatage (DigiByte – DGB).

Nouveautés de la V1.1 :
- Nouveau fichier data.json enrichi
- Image commune "DigiRelik.jpg" utilisée comme visuel de base pour tous les artefacts
- Page hexekta.html : affichage dynamique des artefacts depuis le JSON
- Page art.html : fiche individuelle complète d’un artefact (nom, description, origine, pouvoirs, histoire, ID)
- Intégration de boutons "Mint on XEC" et "Timestamp on DGB"
- Génération automatique de QR code IPFS (placeholder)
- Fichier option.css : tous les nouveaux styles sont ajoutés ici (style.css figé)

Fichiers inclus dans la version :
- index.html
- hexekta.html
- art.html
- artefact.js
- detail.js
- style.css (non modifié)
- option.css (CSS additionnel)
- data.json
- DigiRelik.jpg
- readme.txt

À venir :
- Génération réelle des hashes IPFS
- Intégration des API XEC et DGB
- Système de tri, pagination ou filtrage
- Lore détaillé pour chaque artefact
