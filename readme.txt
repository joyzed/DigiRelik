--------------------

2. Ouvrir index.html comme page principale.
3. hexekta.html contient la galerie.
4. art.html affiche le détail en utilisant ?id= dans l'URL.
5. Les QR codes, fonctions mint, et images spécifiques arriveront en v1.1.1.
